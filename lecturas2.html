ETAPAS EN EL DESARROLLO DE SOFTWARE

El desarrollo de software está conformada por un grupo de etapas, a las cuales se le denominan “Ciclo de Vida”, son el proceso que se llevan a cabo para la construcción de programas informáticos que desarrollan tareas especificas, algunas de las etapas más comunes son:

• Etapa de Análisis: Podemos decir que en esta etapa se encarga de investigar un problema que se quiere resolver, de encontrar la fuente que lo genera o de hallar el sistema que se desea crear, por otro lado también se pueden identificar los componentes que se necesitan para la realización del producto.
• Etapa de Diseño: En esta etapa vemos que su principal tarea es desarrollar el modelo con la información recolectada en la etapa del análisis, o las especificaciones de los componentes del sistema.
• Etapa de Desarrollo o Codificación: En esta etapa se utilizan los modelos que fueron creados en la etapa de diseño, con el objetivo de realizar la creación de los componentes del sistema.
• Etapa de Pruebas: Consiste en asegurar que los componentes individuales del sistema cumplan con los requisitos de las especificaciones que fueron creadas en la etapa de diseño, dando así un buen funcionamiento al sistema.
• Etapa de Implementación: Esta etapa trata de que el sistema que fue diseñado este a disposición del usuario.
• Etapa de Mantenimiento: Como su nombre lo dice, se realiza un mantenimiento al sistema, para solucionar los errores o fallos que el sistema tenga cuando se ejecuta.
• Etapa final EOL (End-of-Life): El fin del ciclo consiste en realizar las tareas necesarias para asegurar que sus clientes estén consientes que el producto no será más comercializado.
